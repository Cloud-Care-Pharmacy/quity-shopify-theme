# Quity Shopify Theme

A modern, feature-rich Shopify theme built for e-commerce stores. This theme provides a comprehensive set of templates, sections, and customization options to create a beautiful and functional online store.

## Theme Information

- **Theme Name:** Modern BD
- **Author:** Brightd data
- **Version:** 1.0
- **Documentation:** [http://brightdata.com.au/](http://brightdata.com.au/)
- **Support Email:** info@brightdta.com.au

## Features

- **Multiple Product Templates:** Various product page layouts including collage, grid, list, stacked, and variant options
- **Flexible Collection Pages:** Multiple collection layouts with different column configurations and filter options
- **Blog Templates:** Grid and list layouts with sidebar options
- **Customizable Settings:** Extensive theme customization through Shopify theme editor
- **Multi-language Support:** Includes translations for multiple languages (Danish, German, English, Spanish, French, Italian, Japanese, Dutch, Portuguese)
- **Responsive Design:** Mobile-first approach ensuring great experience across all devices

## Installation

1. Download or clone this repository
2. Zip the entire theme folder
3. In your Shopify admin, go to **Online Store > Themes**
4. Click **Add theme > Upload zip file**
5. Select the zipped theme folder and upload
6. Once uploaded, click **Publish** to make it your active theme

## Theme Structure

```
quity-shopify-theme/
├── assets/          # CSS, JavaScript, images, and other static assets
├── config/          # Theme settings and configuration files
├── layout/          # Base layout templates (theme.liquid, password.liquid)
├── locales/         # Translation files for different languages
├── sections/        # Reusable theme sections
├── snippets/        # Reusable code snippets
└── templates/       # Page templates (product, collection, blog, etc.)
```

## Customization

This theme includes extensive customization options accessible through the Shopify theme editor:

- **General Settings:** Container width, gutter width, and layout options
- **Product Templates:** Choose from multiple product page layouts
- **Collection Templates:** Various collection page styles and filter configurations
- **Blog Templates:** Different blog post and listing layouts
- **Color Schemes:** Customize colors throughout the theme
- **Typography:** Font selection and styling options

## Templates

### Product Templates
- Default product template
- Collage layouts (s1, s2)
- Grid and list layouts
- Variant options (dropdown, image, image-square)
- Thumbnail positions (left, right, without)
- Tab and accordion descriptions
- Bundle products
- Gift products

### Collection Templates
- 2, 3, and 4 column layouts
- Banner split layout
- Drawer and popup filters
- Infinity scroll and load more options
- List view with color/size options

### Blog Templates
- Grid and list item layouts
- Sidebar options (left, right)
- Multiple layout styles (basic, box, classic, overlay)

## Development

To develop and customize this theme:

1. Install [Shopify CLI](https://shopify.dev/themes/tools/cli)
2. Authenticate with your store: `shopify theme dev`
3. Make your changes locally
4. Preview changes in real-time
5. Push changes when ready: `shopify theme push`
